# MoeKoeMusic 全局自定义动态背景插件       版本：v1.0.3
## 插件简介
为MoeKoe客户端添加全局自定义动态背景，全页面生效，支持自定义透明度、视频更换与清除。

## 功能清单
1. 弹窗设置面板，本地视频上传、预览文件名
2. 文件限制：单文件最大300MB
3. 支持视频格式：mp4 / webm / mov / avi / mkv（5种格式）
4. 0.1~1.0区间透明度滑块调节，保存后全局生效
5. 一键清除背景，重置全部配置

## 安装方式
1. 将全部文件压缩为zip压缩包（不要嵌套文件夹）
2. 打开MoeKoeMusic客户端 → 设置 → 插件 → 安装插件
3. 选中zip包完成安装，点击插件「设置」按钮打开配置面板

## 使用说明
1. 点击「选择视频文件」上传本地视频
2. 拖动滑块调整背景透明度
3. 点击「保存配置」立即生效
4. 更换背景：先点「清除当前背景」，再重新上传视频保存
5. 无视频时客户端恢复默认静态背景
6.配合自定义背景插件，上传静态图片后将背景透明度改为0%，然后保存，即可只有自定义动态背景显示。

## 新增互斥限制说明
本插件与其他任何自定义背景、动态壁纸类插件互斥，无法同时启用：
1. 若已安装其他背景插件，打开插件设置面板会自动禁用全部操作按钮，无法上传/修改视频背景
2. 客户端启动时自动检测冲突，存在冲突时不会渲染视频动态背景
3. 使用前请卸载、禁用其他壁纸/背景类插件，重启客户端即可正常使用

## 独立永久背景更新说明
1. 完全独立渲染，无需安装任何其他自定义背景插件，无第三方插件也能正常显示动态视频背景；
2. 视频二进制永久存储，重启客户端、重启电脑背景配置不会丢失；
3. 存在其他背景插件时自动隐藏本插件背景，防止多层画面叠加；
4. 原有功能全部保留：300MB文件上限、mp4/webm/mov/avi/mkv五种格式、透明度调节、一键清除背景、实时刷新无需重启客户端。

## 互斥BUG修复说明
修复原插件可与其他背景插件多层叠加的问题：
1. 后台每2秒自动扫描全部插件，实时检测壁纸/背景类插件；
2. 一旦检测到冲突，**立刻销毁本插件所有视频背景**，禁止多层叠加；
3. 插件设置弹窗实时刷新冲突状态，冲突时全部按钮锁定，无法保存/上传；
4. 卸载其他背景插件后，背景会自动重新渲染，无需重启客户端。

## 注意事项
1. 超大视频会占用较多内存，建议使用1080p以内短循环视频
2. 修改配置后部分页面需刷新/重启客户端完全生效
3. 视频以base64存储，过大文件会增加客户端缓存占用
